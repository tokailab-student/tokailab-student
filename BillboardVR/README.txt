-----------------------------------------------------------------------------------------------
表示シーンフォルダ
-----------------------------------------------------------------------------------------------
・TEXTURE_WALK_AROUND		　 
		1人が円周に沿って歩くシーン(カメラ8台)

・TEXTURE_WALK_AROUND_blank：
		TEXTURE_WALK_AROUNDからカメラ5のデータだけを消したもの。カメラ5のデータは存在しないため二番目に近いカメラが選ばれる。(カメラ8台)

・TEXTURE_300F				 　	 ：
		TEXTURE_WALK_AROUNDの背景を消していないもの(カメラ8台")

・TEXTURE_Billboard				 ：
		2人が歩きながらボールをパスするシーン(カメラ10台)

・Billboard_WalkThree		   ：
		3人が交差して歩くシーン(カメラ10台)



-----------------------------------------------------------------------------------------------
カメラ位置フォルダ
-----------------------------------------------------------------------------------------------
・Cameras
		撮影カメラ8台分の位置データ。

・CameraPosition
		撮影カメラ10台分の位置データ。



-----------------------------------------------------------------------------------------------
LogSaveフォルダ
-----------------------------------------------------------------------------------------------
・csv
		MainControlerスクリプト内でLogSaveを使用した場合のcsvファイルの格納先。



-----------------------------------------------------------------------------------------------
実行ファイル
-----------------------------------------------------------------------------------------------
実行ファイルの作り方は
		File > Build Setting > Build
Add Open SceneでビルドするSceneの変更ができる。
開発環境がWindowsのためPlatformはPC、Target PlatformはWindowsを選択。


・WALK_AROUND(方法1).exe
		TEXTURE_WALK_AROUNDを使ったシーンを実行ファイルにしたもの(方法1)。

・WALK_AROUND(方法2).exe
		TEXTURE_WALK_AROUNDを使ったシーンを実行ファイルにしたもの(方法2)。

・WALK_AROUND_blank(方法2).exe
		TEXTURE_WALK_AROUND_blankを使ったシーンを実行ファイルにしたもの(方法2)。

・Billboard(方法2).exe
		TEXTURE_Billboardを使ったシーンを実行ファイルにしたもの(方法2)。

・WalkThree(方法2).exe
		TEXTURE_WalkThreeを使ったシーンを実行ファイルにしたもの(方法2)。
